# News Research & Q&A Tool

A lightweight news research and question-answering tool that helps users quickly extract insights from financial and stock market news articles. Users can provide article URLs and ask questions to receive context-aware answers along with source references.

---

## 🚀 Features

- Accepts news article URLs or text files containing URLs  
- Extracts article content using LangChain’s UnstructuredURL Loader  
- Converts text into embeddings using OpenAI embeddings  
- Uses FAISS for fast similarity-based information retrieval  
- Allows users to ask questions and get answers with source URLs  
- Interactive and easy-to-use Streamlit interface  

---

## 🛠 Tech Stack

- Python  
- LangChain  
- OpenAI Embeddings  
- FAISS  
- Streamlit  
- dotenv
